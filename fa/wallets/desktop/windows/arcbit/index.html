ArcBit

ArcBit is designed to be simple and easy to use, while giving users full control over their money. It offers an optional cold wallet storage feature with the ability to authorize payments offline for enhanced security.

ویندوز

نصب کد منبع

کنترل کردن پول خود

این کیف پول کنترل کامل بر بیت کوینها را به شما می دهد. یعنی هیچ شخص سومی نمی تواند موجودی شما را فریز کرده و یا هدر دهد. اما هنوز شما مسئول امنیت و بک آپ گیری از کیف پول خود هستید.

درستی آزمایی متمرکز شده

این کیف پول بطور از پیش قرارداد شده، بر یک سرویس متمرکزشده متکی است. یعنی باید بر طرف سومی اعتماد کرد که وجه های پرداختی را مخفی یا شبیه سازی نکرده باشد.

اَپ راه دور

این کیف پول از راه دور پر می شود. یعنی هر گاه از کیف پول خود استفاده می کنید، باید به توسعه دهندگان اطمینان داشته باشید که بیت کوینهای شما را به دلیل مشکلی در طرف خودشان، ندزدند و یا به هدر ندهند. با استفاده از افزونه های مرورگر یا اَپ های تلفن همراه در صورت در دسترس بودن، می توانید چنین خطری را کاهش دهید.

محیط آسیب‌پذیر

این کیف پول می تواند روی کامپیوترهایی که در برابر بدافزار آسیب پذیر هستند، بارگذاری شود. ایمن کردن کامپیوتر، استفاده از رمز عبور قوی، انتقال بیشتر دارایی تان به محل ذخیره سازی آفلاین و یا فعال کردن تایید دو-فاکتوری می تواند دزدیدن بیت کوینهای شما را دشوارتر سازد.

حریم خصوصی ابتدایی

جلوگیری از جاسوسی روی پرداخت‌های شما

این کیف پول با به گردش درآوردن آدرس ها، جاسوسی روی ترازها و پرداختهای شما را دشوارتر می کند. شما باید همچنان مراقب باشید که هر دفعه که درخواست پرداخت وجه می نمایید، از یک آدرس بیت کوینی جدید استفاده کنید.

افشای اطلاعات برای طرف سوم

این کیف پول از سرورهای مرکزی که می توانند پرداخت های شما را به هم مرتبط کرده و از آی پی آدرس شما یادداشت بردارند، استفاده می کند.

از Tor می توان استفاده کرد

این کیف پول به شما اجازه می دهد Tor را به عنوان یک پراکسی تنظیم کرده و با مرتبط کردن پرداختهای خود با آی پی آدرستان، از آن برای جلوگیری از حمله کنندگان یا تامین کنندگان سرویس اینترنتی استفاده کند.

Dynamic fee with override

This wallet provides fee suggestions which are based on current network conditions which you can override. This means that this wallet will help you choose the appropriate fee so that your transactions are confirmed in a timely manner without paying more than you have to, but ultimately gives you control if you want to override the suggestion.

برای آموزش خود زمان صرف کنید

بیت کوین متفاوت از آن چیزی است که شما می شناسید و هر روز از آن استفاده می کنید. پیش از آغاز استفاده از بیت کوین برای هر تراکنش جدی، مطمئن شوید که آنچه را که باید بدانید را خوانده اید و گامهای درست برای ایمن کردن کیف پول خود را برداشته اید. همواره به یاد داشته باشید که مسئولیت انتخاب دقیق کیف پول و انجام اقدامات مناسب برای حفاظت از پولتان، بر عهده خود شماست.
