KeepKey

KeepKey is a hardware wallet that makes bitcoin security simple. When you entrust KeepKey with your money, every bitcoin transaction you make must be reviewed and approved via its OLED display and confirmation button.

سخت‌افزار

مشاهده وب سایت کد منبع

کنترل کردن پول خود

این کیف پول کنترل کامل بر بیت کوینها را به شما می دهد. یعنی هیچ شخص سومی نمی تواند موجودی شما را فریز کرده و یا هدر دهد. اما هنوز شما مسئول امنیت و بک آپ گیری از کیف پول خود هستید.

درستی آزمایی متغیر

ویژگی های درستی آزمایی پرداخت وجه به وسیله کیف پول نرم افزاری که با این دستگاه استفاده می کنید، فراهم می شود. خواهشمند است امتیاز درستی آزمایی کیف پول نرم افزاری که قصد استفاده از آنرا دارید، نگاه کنید.

اَپ جدید

این کیف پول هنوز آزمایش نشده و تعدادی زیادی از کاربران آنرا بطور عمومی بازبینی نکرده اند. یعنی این اَپ بسیار ممکن است کد خطرناکی را پنهان کرده باشد یا کاری انجام دهد که دلخواه شما نباشد.

محیط بسیار امن

این کیف پول از محیط امن ویژه ای که توسط این دستگاه فراهم آمده، پر می شود. چون هیچگونه نرم افزاری در این محیط نمی تواند نصب شود، بنابراین حفاظت خوبی در برابر حملات و بدافزارهای کامپیوتری ایجاد شده است.

حریم خصوصی متغیر

نرم افزار کیف پولی که با این دستگاه از آن استفاده می کنید، ویژگی های حریم خصوصی را فراهم می کند. خواهشمند است امتیاز حریم خصوصی را برای کیف پول نرم افزاری که قصد استفاده از آن را دارید، ملاحظه نمایید.

Variable fee control

Fee control features are provided by the software wallet you use with this device. Please see the Fee Control score for the software wallet you plan to use.
