GreenAddress

GreenAddress ist eine nutzerfreundliche Multi-Signatur Wallet mit erhöhter Sicherheit und Privatsphäre. Zu keinem Zeitpunkt werden Schlüssel auf Servern gespeichert. Aus Sicherheitsgründen sollten Sie immer Zwei-Faktor Authentifizierung sowie die Browser-Erweiterung oder die Android App verwenden.

Geteilte Konrolle über ihr Geld

Diese Wallet setzt für jede Transaktion vorraus, dass diese durch Sie und den Dienst authorisiert wird. Unter normalen Umstanden können Sie über das ursprünglichen Backup oder eine vorab gezeichnete Transaktion per Email die Kontrolle über ihre Bitcoins wiederbekommen.

Dezentralisierte Validierung

Diese Wallet verbindet sich mit einem zufälligen Server aus einer Liste. Das bedeutet, dass wenig Vertrauen in Dritte beim Bestätigen von Zahlungen nötig ist. Dennoch ist es nicht so sicher wie ein "Full Node" wie Bitcoin Core.

Grundlegende Transparenz

Die Entwickler dieser Wallet veröffentlichen den Quellcode für den Client. Das bedeutet, dass jeder Entwickler auf der Welt den Code begutachten kann. Dennoch; Sie müssen immernoch den Entwicklern vertrauen wenn Sie die finale Software installieren oder updaten, da sie nicht determinisitisch gebaut wurde so wie Bitcoin Core.

Zwei-Faktor Authentifizierung

Diese Wallet kann in unsicheren Umgebungen verwendet. Allerdings wird eine Zwei-Faktor Authenifizierung vorausgsetzt. Das bedeutet, dass Zugang zu mehreren Geräten oder Accounts nötig ist um Bitcoins zu stehlen.

Grunglegende Privatsphäre

Verhindert das Ausspionieren ihrer Zahlungen

Diese Wallet macht es durch das Verwenden von wechselnden Adressen schwer ihren Kontostand oder Zahlungen auszuspionieren. Sie sollten immer noch dafür sorgen eine neue Bitcoin Adresse zu verwenden jedes Mal wenn Sie eine Zahlung anfragen.

Legt Informationen für Dritte offen

Dieser Service kann ihre Zahlungen miteinander in Verbindung bringen, ihre IP Adresse loggen und kennt ihre wahre Indentität falls Sie private Informationen wie Emails, Namen oder Bankdaten zur Verfügung stelllen.

TOR wird nicht unterstützt

Diese Wallet lässt Sie Tor, mit dem man Angreifer oder Internetanbieter davon hätte abhalten können ihre Zahlungen mit ihrer IP Adresse in Verbindung zu bringen, nicht verwenden.

Full control over fees

This wallet gives you full control over fees. This means that this wallet allows changing the fees after funds are sent using RBF or CPFP. This wallet also provides fee suggestions based on current network conditions so that your transactions are confirmed in a timely manner without paying more than you have to.
