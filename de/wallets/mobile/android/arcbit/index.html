ArcBit

ArcBit is designed to be simple and easy to use, while giving users full control over their money. It offers an optional cold wallet storage feature with the ability to authorize payments offline for enhanced security.

Kontrolle über Ihr Geld.

Diese Wallet gibt ihnen volle Kontrolle über ihre Bitcoins. Das bedeutet, dass kein Dritter ihr Vermögen verlieren oder einfrieren kann. Dennoch; Sie sind selbst dafür verantwortliche ihre Wallet zu sichern und Backups zu erstellen.

Zentralisierte Validierung

Diese Wallet vertraut standartmäßig auf einen zentralisierten Service. Das bedeutet, dass Dritten vertraut werden muss, dass diese keine Zahlungen verstecken oder simulieren.

Grundlegende Transparenz

Die Entwickler dieser Wallet veröffentlichen den Quellcode für den Client. Das bedeutet, dass jeder Entwickler auf der Welt den Code begutachten kann. Dennoch; Sie müssen immernoch den Entwicklern vertrauen wenn Sie die finale Software installieren oder updaten, da sie nicht determinisitisch gebaut wurde so wie Bitcoin Core.

Sichere Umgebung

Diese Wallet wird von Smartphones geladen bei denen Apps normalerweise isoliert sind. Das bietet guten Schutz vor Malware, obgleich Smartphones häufiger verloren gehen oder gestohlen werden. Das Smartphone zu verschlüsseln und Backups von ihrer Wallet zu erstellen kann das Risiko reduzieren.

Grunglegende Privatsphäre

Verhindert das Ausspionieren ihrer Zahlungen

Diese Wallet macht es durch das Verwenden von wechselnden Adressen schwer ihren Kontostand oder Zahlungen auszuspionieren. Sie sollten immer noch dafür sorgen eine neue Bitcoin Adresse zu verwenden jedes Mal wenn Sie eine Zahlung anfragen.

Legt Informationen für Dritte offen

Diese Wallet nutzt zentrale Server welchen es möglich ist ihre Zahlungen miteinander in Verbindung zu bringen und ihre IP Adresse zu loggen.

TOR wird nicht unterstützt

Diese Wallet lässt Sie Tor, mit dem man Angreifer oder Internetanbieter davon hätte abhalten können ihre Zahlungen mit ihrer IP Adresse in Verbindung zu bringen, nicht verwenden.

Dynamic fee with override

This wallet provides fee suggestions which are based on current network conditions which you can override. This means that this wallet will help you choose the appropriate fee so that your transactions are confirmed in a timely manner without paying more than you have to, but ultimately gives you control if you want to override the suggestion.
