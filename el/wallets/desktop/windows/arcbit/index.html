ArcBit

ArcBit is designed to be simple and easy to use, while giving users full control over their money. It offers an optional cold wallet storage feature with the ability to authorize payments offline for enhanced security.

Εγκατάσταση Πηγαίος κώδικας

Έλεγχος των χρημάτων σας

Το πορτοφόλι αυτό σας δίνει πλήρη έλεγχο των bitcoin σας. Αυτό σημαίνει ότι κανένας τρίτος δεν μπορεί να παγώσει ή να χάσει τα κεφάλαιά σας. Είστε ωστόσο ακόμα υπεύθυνος για την διασφάλιση και τη δημιουργία αντιγράφων ασφαλείας του πορτοφολιού σας.

Κεντρική επικύρωση

Το πορτοφόλι αυτό βασίζεται σε μια κεντρική υπηρεσία από προεπιλογή. Αυτό σημαίνει ότι ένας τρίτος πρέπει να είναι έμπιστος ότι δεν θα αποκρύπτει ή θα προσποείται πληρωμές.

Απομακρυσμένη εφαρμογή

Το πορτοφόλι αυτό φορτώνεται από απομακρυσμένη τοποθεσία. Αυτό σημαίνει ότι οποτεδήποτε χρησιμοποιείτε το πορτοφόλι σας, πρέπει να εμπιστεύεστε τους προγραμματιστές ότι δεν θα κλέψουν ή θα χάσουν τα bitcoins σας σε ένα περιστατικό στην ιστοσελίδα τους. Χρησιμοποιώντας μια επέκταση στο πρόγραμμα περιήγησης ή μια εφαρμογή κινητού τηλεφώνου, εφόσον υπάρχουν, μπορoύν να μειώσουν αυτό τον κίνδυνο.

Ευάλωτο περιβάλλον

Το πορτοφόλι αυτό μπορεί να φορτωθεί σε υπολογιστές οι οποίοι είναι ευάλωτοι στο κακόβουλο λογισμικό. Ασφαλίζοντας τον υπολογιστή σας, χρησιμοποιώντας μια δυνατή φράση πρόσβασης, μεταφέροντας τα περισσότερα από τα κεφάλαια σας για "αποθήκευση στην κατάψυξη" (cold storage) ή επιτρέποντας έλεγχο ταυτότητας δύο παραγόντων μπορεί να κάνει πιο δύσκολη την κλοπή των bitcoin σας.

Βασική προστασία ιδιωτικότητας

Αποτρέπει κατασκοπεία των πληρωμών σας

Το πορτοφόλι αυτό καθιστά δυσκολότερη την κατασκοπεία του υπολοίπου σας και των πληρωμών μέσω εναλλασσόμενων διευθύνσεων. Θα πρέπει ακόμα να φροντίσετε να χρησιμοποιείτε μια νέα διεύθυνση Bitcoin κάθε φορά που κάνετε αίτημα πληρωμής.

Αποκαλύπτει πληροφορίες σε τρίτους

Το πορτοφόλι αυτό χρησιμοποιεί κεντρικούς διακομιστες οι οποίοι μπορούν να συσχετίσουν τις πληρωμές σας μαζί και να συνδεθούν στη διεύθυνση ΙΡ σας.

Το Tor μπορεί να χρησιμοποιηθεί

Το πορτοφόλι αυτό σας επιτρέπει να εγκαταστήσετε και να χρησιμοποιήσετε το Tor ως ένα διακομιστή μεσολάβησης (proxy) για να αποτρέψετε εισβολείς ή παρόχους υπηρεσιών Διαδικτύου από το να συσχετίζουν τις πληρωμές σας με τη διεύθυνση IP σας.

Dynamic fee with override

This wallet provides fee suggestions which are based on current network conditions which you can override. This means that this wallet will help you choose the appropriate fee so that your transactions are confirmed in a timely manner without paying more than you have to, but ultimately gives you control if you want to override the suggestion.

Αφιερώστε χρόνο για να εκπαιδευθείτε

Το Bitcoin είναι διαφορετικό απ' ό,τι γνωρίζετε και χρησιμοποιείτε κάθε μέρα. Πριν αρχίσετε να χρησιμοποιείτε το Bitcoin για σοβαρές συναλλαγές, να διαβάσετε πρώτα ό,τι χρειάζεται να γνωρίζετε και να κάνετε τα απαραίτητα βήματα ώστε να διασφαλίσετε το πορτοφόλι σας. Να θυμάστε πάντα ότι είναι δική σας ευθύνη να επιλέξετε προσεκτικά το πορτοφόλι σας και να υιοθετήσετε τις σωστές πρακτικές προκειμένου να προστατεύσετε τα χρήματά σας.
