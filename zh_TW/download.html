你需要保持耐心

Bitcoin Core 初始同步會需要較長時間。請確保您有足夠的頻寬和儲存容量來滿足完整的區塊鏈大小 (超過 65GB) 。如果您知道如何下載一個 torrent 文件，您可以在開始運行這個軟體前通過把bootstrap.dat (一個區塊鏈的先前備份) 放到 Bitcoin Core 的數據目錄中來加快同步進程。 Read the full node guide for details.
